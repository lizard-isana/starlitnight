about

"Starlit Night" is an online planetarium written in JavaScript.
This project is currently under development. Some bugs are expected and some modifications will be made.

Features

  • Written in JavaScript, no plug-ins required.
  • Hemispherical starmap by perspective projection.
  • Scrolling & zooming by mouse dragging.
  • Stars, planets, sun, moon & constellations
  • Moon phase
  • Labels for stars, planets & constellations
  • Asterisms of the constellations
  • Azimuth & elevation grids
  • Time & place settings
  • Permalink
  • User defined markers
  • Visible passes of artificial satellites

Requirements

  • Firefox3, Safari3, Chrome10, IE9 or later
  • 640pixel X 480pixel
